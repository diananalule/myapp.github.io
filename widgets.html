The flow of how the app is as the screenshots and their descriptions on them

Login Page

Here, the user will have to enter their detailswith which they signed up
before accessing the app.

Sign up Page

Here the user will register their personal details.

Profile Page

Details of a person who has registered.

lost objects page

By scrolling through this view a user is able to view all lost items.
And if at all their item is in the can continue with the procedure of retrieval

Picker Page

Here the user must register the found item by choosing the category where it falls and give a little description about the found item.

Post Page

Here the user must register the lost item by choosing the category where it falls and give a little description about the lost item.

Found Page

By scrolling, the logged in user is able to view found items
posted by other different users

Chat Page

Here the person who lost an item can chat with the person who found the item,
negotiate on where and how to recieve the lost item .
